# Activation Redesign for a B2B Workflow Product

## Problem

Activation was underperforming because new users faced too much setup friction before reaching first value. The team had ideas, but no shared view of the most important bottleneck.

## Context / Role

Led product framing and aligned product, design, operations, and engineering around one activation metric and one focused release plan.

## Solution / Approach

- Mapped the funnel to isolate where users stalled.
- Prioritized a two-sprint package of guided setup, templates, and clearer empty states.
- Added lightweight experiment tracking to tie release choices to outcome metrics.

## Impact

- Activation rate improved by 11 percentage points.
- Time to first value dropped by 18 percent.
- Release moved from diagnosis to production in two sprints.

## Key Learnings

- Shared definitions accelerate delivery more than extra debate.
- Smaller, sharper releases often create more measurable progress than broader redesigns.
